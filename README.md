### React boilerplate
Simple React boilerplate with webpack.

### Installation
Step 1: If you haven't installed webpack already, install it first, otherwise go to step 2.

```sh
$ npm install webpack -g
```
Step 2: install the node modules.
```sh
$ npm install
```
### Tasks


Development:

```sh
$ npm start
```
Production:
```sh
$ npm run deploy
```
